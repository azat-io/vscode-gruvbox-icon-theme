This extension includes Gruvbox style icons for Visual Studio Code. The main focus in the development of the extension is paid to matching the colors of the icons to the original Gruvbox theme for syntax highlighting.

The extension supports both light and dark themes.

> [!TIP]
> I'm now working on [Eyecons](https://github.com/azat-io/eyecons) — a new, more advanced icon theme for VS Code. Check it out!
